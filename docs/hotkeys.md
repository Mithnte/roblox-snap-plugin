# Hotkeys

- 1 — Select
- 2 / M — Move (axis lock: X/Y/Z)
- 3 / R — Rotate (Q/E rotate; hold X/Y/Z to pick the axis)
- 4 — Scale (=/- to grow/shrink; hold X/Y/Z to constrain an axis)
- B — Box Select
- J/L/I/K/U/O — Align sides (to first)
- C — Align center to first
- T — Match rotation to first
- S — Match size to first (BasePart)
- G — Toggle Grid Snap
- F — Toggle Rotate Snap
- V — Toggle Vertex/Edge Snap
- [ / ] — Cycle Grid Step (global)
- Ctrl+Shift+D — Smart Duplicate (repeats your last Move offset)
- Ctrl+P — Command Palette

## Tool switching is global

Tool switching (1/2/3/4/B), the snap toggles (G/F/V) and grid-step cycling
([ / ]) are handled as global shortcuts, so they work no matter which tool is
currently active — you can jump straight from Move to Rotate, for example.

## Rebinding
1/2/3/4, B, G, F and V are rebindable from the panel's "Hotkeys" section: click
a row, then press the new key. Bindings are saved and persist across Studio
restarts. Contextual/combo keys (Q/E, =/-, J/L/I/K/U/O, X/Y/Z axis lock,
Ctrl+Shift+D, Ctrl+P) are fixed to avoid conflicts.
